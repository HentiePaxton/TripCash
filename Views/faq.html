What Is TripCash?

Tripcash is a form of digital currency which is based on an open source code that was created and is held electronically. Tripcash is a decentralized form of travelling, meaning that it does not belong to any form of government and is not controlled by anyone, and can be used for all your traveling needs.

How Do TripCash Transactions Work?

From an end user side, using TripCash is extremely simple. It can be based on software on your computer or an application on your mobile device. You use the software to purchase goods or services from “hosts”, example being someone renting out accommodation or offering transport services. The wallet has a unique encrypted set of numbers and letters which is linked only to your software and no one else’s. Every transaction is publicly shared on the TripCash network on a ledger labeled the “block chain”. This ledger contains every single transaction ever made, which gives complete transparency for each transaction and allows any computer on the network to verify its validity. A digital signature is linked to each transaction, which corresponds with the sending address.

Can You Lose TripCash?

Since TripCash are stored in your software wallet, it isn’t possible for you to lose the TripCash. However, you can lose the wallet which contains your TripCash currency. If this happens, then the TripCash are simply out of circulation. When a wallet is lost, the missing TripCash are offset by the law of supply and demand. The missing TripCash will increase the value of the remaining TripCash, resulting in compensation for those that have been lost.

How Is It Possible For TripCash To Hold Value?

Simply put, TripCash holds value because it can be used as money to pay for travel, or provide services for travellers. Just like any other currency, TripCash value is greatly influenced by who uses the currency. The hosts, businesses, and users determine the value of TripCash by choosing to use TripCash over other currencies. Simply put, the more people who choose to use TripCash as a form of payment, the greater the value of each TripCash will be.

The second way to get TCH would be to make a purchase through exchange. Generally, there are plenty of different exchange sites available.
